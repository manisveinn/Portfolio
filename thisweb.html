This website

I created this website for an assignment at the University of Iceland. Here is how I did it:
In the summer of 2019, I was offered the assignment of supervising the website of a company called Bílanaust. At the time I thougt it meant that I needed to learn HTML. Therefore I spent a few days at work learning HTML and during that time I made a layout for a site after a youtube tutorial from a youtube channel called mmtuts. After all I didn't have to know HTML to supervise the website at work, since it had a CMS controlling it (not a very good one if you ask me).
So when I got the assignment of making a portfolio website for a class at the University, I simply dug up the layout I had made the summer before and adapted and improved it with the help of two more tutorials from the same youtube channel.
